# Горячие клавиши 
#### Выделить всё:
Ctrl+A
#### Копировать:
Ctrl+C
#### Вставить:
Ctrl+V
#### Сохранит:
Ctrl+S
#### Вырезать:
Ctrl+X
#### Отменить:
Ctrl+Z

# Работа с git 
### скачать проект:
```bash
git clone <URL>
```
### учесть все файлы:
```bash
git add .
```
### Сохранить коммит:
```bash
git commit -m "..."
```
### отправка в облако:
```bash
git push
```
